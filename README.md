# Job Tracker Application

En enkel og moderne React-app for å holde oversikt over jobbsøknader.

## ✨ Funksjoner

- Legg til jobbsøknader
- Endre status (Sendt, Intervju, Avslag, Tilbud)
- Filtrer søknader
- Se oversikt over aktive, avslag og tilbud
- Lagres automatisk i localStorage
- Sorterer nyeste først

## 🛠 Teknologi

- React
- Vite
- CSS
- LocalStorage

## 🚀 Hvordan kjøre prosjektet

```bash
npm install
npm run dev
